# NotionAPINode
 - express server
 - notion API Test
 - Database (Calendar) Query Test
  
## Before start node.js
- set your own .env configuration (secret_key & database_id)
- make sure your Notion database is connected with your API integration.
- install node_modules
```
 npm install
```
or
```
 yarn
```

## start node.js

```
npm start
```
or
```
yarn start
```
